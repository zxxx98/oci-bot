OCI 配置初始化

OCI 初始配置从哪里获取

页面中的 OCI 初始化需要 `Tenancy OCID`、`User OCID`、`Fingerprint`、`Region` 和 `Private Key` 这 5 项。

  • Tenancy OCID：在 OCI 控制台右上角头像菜单进入租户页面后查看。
  • User OCID：在 `Identity & Security` -> `Users` -> 当前用户详情页中查看。
  • Fingerprint：在当前用户的 `API Keys` 中上传公钥后获得。
  • Region：填写你准备抢机的区域，例如 `ap-singapore-1`。
  • Private Key：填写与你上传公钥配对的私钥内容。

如果你还没有 API Key，可以先在本地生成：

openssl genrsa -out oci_api_key.pem 2048
openssl rsa -pubout -in oci_api_key.pem -out oci_api_key_public.pem

然后把公钥上传到 OCI 用户的 `API Keys`，再把私钥内容粘贴到页面里，最后点击“验证 OCI 配置”。
